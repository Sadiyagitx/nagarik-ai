# Nagarik AI

Nagarik AI is an AI-powered election awareness and citizen assistance web platform designed to help users understand voting processes, check eligibility, and access civic information through an interactive AI assistant.

The platform focuses on improving civic engagement by making election-related information simple, accessible, and interactive for citizens.

---

## Features

- AI-powered citizen assistant
- Election and voting awareness system
- Voting eligibility checker
- Step-by-step voting guidance
- Election timeline information
- Interactive chatbot interface
- English and Hindi language support
- Modern responsive UI
- Share functionality for awareness results
- Dynamic and user-friendly experience
